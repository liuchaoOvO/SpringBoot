# Spring-Boot
SpringBoot 的多模块项目代码
内容包括：

1，整合日志

2，整合mybatis

3，整合Redis

4，整合RabbitMQ

5，整合thymeleaf模板

6，整合Spring Security

7，加配置文件关键信息的加密，加验证码，加二维码的生成

8，热部署，druid数据源，pagehelper分页插件，fastjson模板

9，script_sql 存放springboot.sql  完整的sql  表结构和测试数据

目前模块分为：
<module>springboot-demo</module>
<module>springboot-security</module>
